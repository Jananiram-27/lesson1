GitHub Commands

  • Git init

    • Git init command is used to initialize local and global git repository

  • Git add .

    • Git add . command is used to add the files in to the git repository

  • git commit -m "message"

    • This command is used to commit the file by displaying message

  • git status

    • This command is used to show the status of current commit message

  • git branch

    • This command is used to know which branch whether it is master or main branch

  • git push origin main

    • This command is used to push the files in to a global repository
